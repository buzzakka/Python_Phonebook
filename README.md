## Установка
Установка осуществляется из корневой папки проекта, в которой находится Makefile.

### Установка с запуском приложения
#### Через Makefile
```bash
make all
```
#### Через терминал вручную
```bash
python3 -m venv venv
venv/bin/pip3 install -r requirements.txt
./venv/bin/python3 app.py
```
